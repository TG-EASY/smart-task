# Smart Task Manager

A simple static task manager app built with HTML, CSS, and JavaScript.

## Deployment

This project is a static site and can be deployed on Vercel without a build step.

### Vercel settings

- Root Directory: `./`
- Framework: `Other`
- Build Command: (leave blank)
- Output Directory: (leave blank)
- Install Command: (leave blank)

## Usage

Open `index.html` in a browser or deploy the repository to Vercel.
